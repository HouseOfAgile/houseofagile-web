HeyPet

Social network for pet owners with real-time messaging, location-based services, and community features.

The Challenge

Pet owners struggled to find trusted sitters and coordinate care reliably. Existing solutions lacked real-time communication and trust signals needed to build confidence between owners and carers.

The Solution

Built a marketplace platform focused on real-time communication, smart matching, and a streamlined booking flow. Implemented robust profiles, reviews, and messaging to establish trust and accelerate conversions.

Technical Implementation

Backend Architecture

  • Symfony REST API
  • PostgreSQL Database
  • OAuth 2.0 Authentication
  • Docker Containerization

Frontend Stack

Key Features

  • Location-based Services

  • Community Forums
